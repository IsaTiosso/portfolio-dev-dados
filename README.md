## ✨ Sobre este Portfólio

Este repositório reúne projetos desenvolvidos com foco em **análise, interpretação e visualização de dados**, explorando ferramentas e metodologias amplamente utilizadas na área.

O objetivo é demonstrar:

- raciocínio analítico;
- organização de projetos;
- capacidade de transformar dados em informação útil;
- evolução prática no uso de ferramentas de dados.

Cada projeto foi pensado para fortalecer habilidades técnicas e construir um portfólio cada vez mais sólido e estratégico.
